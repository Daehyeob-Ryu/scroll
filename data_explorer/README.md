# Data Explorer

A web-based data search and exploration platform.

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

## Features

-   **Large Dataset Support**: Efficiently handles large lists of data.
-   **Advanced Search**: Search by keyword, ID, category, or tags.
-   **Filtering**: Filter by Category and Status.
-   **Detail View**: Click on any row to view detailed information.
-   **Dark Mode**: Built with a professional dark theme.
